# Recognition of Mathematical Expressions from Document Images Using Encoder–Decoder Models
## Evaluation of trained models

Evaluation outputs `evaluation.csv`. This file contains results of evaluation of trained model on target validation dataset. For all outputs following values are computed:

- CER
- Model's confidence 
- Mean sequence probability
- Minimum sequence probability
- Mean sequence entropy
- Mean sequence probablity with latex check
- Probability variance
- Low confidence ratio
- Combined score

output csv file head:

- text = output generated by model
- ground truth = correct value for input image
- CER = character error rate
- Posterior probabilty = model's confidence of generated output
- Mean sequence prob = mean probability of all tokens in output sequence
- Min sequence prob = minimum probability of all tokens in output sequence
- Mean sequence entropy = mean entropy of output tokens
- Mean sequence probablity with latex check = mean probability of all tokens in output sequence but set to zero of the generated latex is invalid
- Probability variance = variance of all tokens in output sequence
- Low confidence ratio = percent of token probabilities smaller than $0.5$
- Combined score = custom metric combining Mean sequence prob, Mean sequence entropy, Low confidence ratio and latex check

Script was ran using following setup:

- Python 3.10.12
- CUDA Version: 13.0 
- NVIDIA Driver Version: 580.126.18

(older CUDA or NVIDIA Driver versions can cause PyTorch/Transformers library incompatibilities)

1. To download model and dataset, run 

```
cd evaluation
./install.sh
```

2. Run evaluation script

```
./run.sh <model_folder> <output.csv> <eval>
```

where

- `model_folder` is folder on huggingface_hub (see https://huggingface.co/xklimo04/MER/tree/main)
- `output.csv` is name of required output file
- `eval` is dataset to be evaluated where possible options are: `valid` or `test`

To evaluate the current best model `self_training_second_stage_0.1-1` on test set run:
```
./run.sh self_training_second_stage_0.1-1 output.csv test
```


For plotting the outputs, run `confidence_measure_comparison.py` to see Prediction Power of Confidence Measures

## Training the baseline model
To replicate the training:

1. Load the dataset:
```
python3 src/prepare_non_target_dataset.py
```

2. Pretrain on non-target dataset:
```
python -m src.pretrain_on_non_target \
    --batch_size 16 \
    --lr 5e-5 \
    --epochs 40 \
    --project_dir . \
    --output_dir results_end_to_end \
    --num_beams 4 \
    --max_length 256 \
    --early_stopping \
    --augment
```
3. Download the taget dataset from https://huggingface.co/datasets/Veronika02/target/tree/main. You can for example use the `evaluation/install.sh` script.

4. Finetune on target dataset:
```
python -m src.finetune_on_target \
    --batch_size 8 \
    --lr 1e-5 \
    --epochs 10 \
    --project_dir . \
    --output_dir results_finetuned_end_to_end \
    --num_beams 4 \
    --max_length 256 \
    --early_stopping \
    --target_path evaluation/valid/valid/dataset/target \
    --model_dir results_end_to_end/best_model
```
